== Installation ==

1. Upload 'b3-editor-override' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize the toolbars on the settings screen

== Frequently Asked Questions ==

= Q. Does it work with the Gutenberg editor ? =
A. No, this is for the classic editor.
